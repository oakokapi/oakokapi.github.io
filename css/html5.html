Green Purpose Internship

Business Description

Green Purpose, LLC is an eco-minded enterprise that strives to reduce the amount of valuable resources that end up in landfills on a daily basis. We do this by reusing materials, repurposing materials and recycling materials. We are currently seeking web design interns to add to our team.*

Job Responsibilities

You will be responsible for assisting the design and graphic production of our webpage. You must work as a team with the Senior Web Designers, take design concepts, apply them over a range of products and create optimized files and specs for the developers. Interested applicants should have some knowledge of the following:

  • Web design
  • PHP
  • MySQL
  • Graphic Design
  • CSS
  • HTML
  • Running servers
  • Web hosting
  • Internet/network security
  • Mobile development
  • iPhone/Android OS

How To Apply

Please send resumes and cover letters to the Director of Business Development Green Purpose.
